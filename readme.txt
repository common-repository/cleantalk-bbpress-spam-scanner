Check existing bbPress topics for spam and move to trash all found spam.

== Description ==

CleanTalk bbPress spam scanner is an add-on for <a href="https://wordpress.org/plugins/cleantalk-spam-protect/">CleanTalk Anti-Spam Plugin</a> and extends its capabilities. The plugin is designed to help you find and move to trash all existing bbPress spam topics.

The plugin scans all bbPress topics and checks the author parameters through the CleanTalk database. After scanning, you will receive a list of all found spam topics and you can choose which of them to move to the trash or move them all at once.

== Installation ==

Please, note that the Anti-Spam by CleanTalk plugin and bbPress plugin should be activated.

1. Download, install and activate 'CleanTalk bbPress spam scanner'.

2. The scanner you can find on the Comments submenu page.
